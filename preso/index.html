Backbone.js

2.1 What does Backbone.js contain?

Backbone.js is a small collection of classes and utility functions. In this presentation we will cover the basics: Model, Collection, View and Events while building a simple demo application.

  • Model - Holds data and business logic
  • Collection - Holds a ordered sets of models
  • View - UI and presentation logic
  • Events - Mixin to provide an object with pub/sub ability
  • History - Global router to handle hashchange and pushstate events
  • Router - Maps urls to functions
  • Sync Function - Syncronizes models with data stores
  • Utility Functions - noConflict() and setDomLibrary()

4.2 Additional Functions

                


                

                    

                        

                            
Underscore.js

                            
Backbone.js proxies to Underscore.js the following 28 functions:

                                
                                forEach (each), map, reduce (foldl, inject), reduceRight (foldr),
                                find (detect), filter (select), reject, every (all), some (any),
                                include, invoke, max, min, sortBy, groupBy, sortedIndex, shuffle,
                                toArray, size, first, initial, rest, last, without, indexOf, lastIndexOf,
                                isEmpty, chain
                                
                            

                            
Additional

                            
Backbone.js addes additional functions to manage the array of models:

                                
                                at, push, pop, unshift, shift, length, comparator, sort, pluck, where
